# WordPress Plugin Assets

This directory is used by the WordPress Plugin Repository to display images for your plugin.

## Required Files

### Icons
- `icon-128x128.png` (Small icon)
- `icon-256x256.png` (Large icon)
- `icon.svg` (Optional, vector icon)

### Banners
- `banner-772x250.png` (Normal banner)
- `banner-1544x500.png` (High DPI banner)

### Screenshots
- `screenshot-1.png`
- `screenshot-2.png`
- ...and so on.

The filenames must match exactly as shown above. Screenshots should correspond to the `== Screenshots ==` section in your `readme.txt`.
